# 📊 E-Commerce Sales Data Analytics (Superstore Dataset)

## 📌 Project Overview
This project focuses on analyzing E-Commerce sales data using the Superstore dataset. The goal is to extract meaningful insights from raw transactional data and visualize business performance using Python.

The system performs data preprocessing, exploratory data analysis (EDA), and generates visualizations to understand sales trends, product performance, and customer behavior.

---

## 📂 Dataset
- **Name:** Superstore Dataset  
- Contains sales transaction data including:
  - Order Date
  - Sales
  - Profit
  - Category & Sub-Category
  - Customer Details
  - Region

---

## ⚙️ Technologies Used
- Python 🐍  
- Pandas  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## 🔄 Project Workflow
1. Data Ingestion (Load CSV dataset)
2. Data Cleaning (Handle missing values, duplicates)
3. Data Preprocessing (Convert data types, feature extraction)
4. Exploratory Data Analysis (EDA)
5. Data Visualization
6. Insights Generation

---

## 📊 Visualizations Generated
- 📈 Monthly Sales Trend  
- 📊 Yearly Sales Comparison  
- 🏆 Top 10 Products by Sales  
- 💰 Top 10 Products by Profit  
- 📦 Sales by Sub-Category  
- 🔥 Monthly Sales Heatmap  
- ⚖️ Profit vs Sales Scatter Plot  

---

## 📈 Key Insights
- Sales show strong growth over the years, especially in 2017  
- Peak sales occur during year-end (Nov–Dec)  
- Phones and Chairs are top-performing categories  
- A small number of products contribute to most revenue (Pareto Principle)  
- Some high sales transactions generate low profit
